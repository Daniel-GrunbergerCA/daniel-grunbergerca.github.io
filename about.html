# Adding a new post is a two-step process:

1. Create posts/your-slug.md and write your post in Markdown.
2. Add an entry to posts.json:

  {
    "slug": "your-slug",
    "title": "Your Post Title",
    "date": "2026-04-14",
    "tags": ["ctf", "htb"],
    "excerpt":"Short description shown on the home page."
  }

3. git push — live on GitHub Pages within seconds.
